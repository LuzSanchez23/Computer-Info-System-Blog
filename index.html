Application Layer


The application layer, ensures effective communication with another application program on a network. This layer is used by end-user software such a web browsers and emails clients. It provides protocols that allow software to send and received information and present meaningful data to users. Enables, if appropiate, authentication to occur between devises for an extra layer of security. Application layer is the hightest level of open systems, providing services directly for the application process.

  • Protocols
  • Domain Name System
  • Hypertext Transfer Protocol
  • Simple Mail Transfer Protocol
  • Post Office Protocol
  • Dynamic Host Configuration Protocol
  • File Transfer Protocol
  • Internet Message Access Protocol
